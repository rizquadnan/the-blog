# How to run locally (for developers)
## Prerequiste
1. Install node.js
2. Install npm
3. Install yarn
4. Clone and run this repo locally (https://github.com/rizquadnan/the-blog)

## Steps to run app
1. Clone this repo
2. On terminal execute: `yarn`
3. On terminal execute: `yarn start`
